## 🔎 About

### This repository

This repository provides solutions to CTFs (*Capture the flag*) from various platforms. It covers exploitation techniques, vulnerabilities, tooling and methodology. Intended as both a personal reference and public resource.

**Environment:** The default OS you will find in these challenges is **Kali Linux**
<img src="https://cdn.simpleicons.org/kalilinux/557C94" width="24" height="24" alt="Kali Linux" style="vertical-align: middle;" />
